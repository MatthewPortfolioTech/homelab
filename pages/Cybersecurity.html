Cybersecurity Fundamentals Lab

Hands-on security lab focused on firewall configuration, network segmentation, vulnerability scanning, threat detection, and system hardening — simulating real defensive security operations.

// overview

Project Details

This lab simulates a company's cybersecurity environment. Rather than just studying theory, each component is configured and tested in a real virtual environment — including firewall rules, network segmentation, vulnerability assessments, event monitoring, and security policy implementation.

The lab builds directly on the networking and infrastructure knowledge from previous projects, applying a defensive security layer on top of a working enterprise environment.
