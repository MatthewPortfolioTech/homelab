Project Details

This project will focus on eliminating repetitive manual work through scripting and automation. Examples include automated user provisioning in Active Directory, system health monitoring, log parsing, and bulk configuration deployments across multiple machines.

Scripts will be well-commented and documented so they can be understood, adapted, and reused — the same standard expected in a real IT environment.
