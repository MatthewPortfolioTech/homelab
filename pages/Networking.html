Networking Lab

Hands-on network setup with real hardware and simulators — VLANs, IP routing, switching, and connectivity testing between servers and clients.

// overview

Project Details

This project covers both virtual network simulation using Cisco Packet Tracer and GNS3, and a physical home lab built with real managed switches. The goal is to understand how enterprise networks are designed, configured, and troubleshot in practice.

I'm working through foundational concepts first — topology design, IP addressing, VLANs — before moving into advanced routing, security, and firewall integration.
