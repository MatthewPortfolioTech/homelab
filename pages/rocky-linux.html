Rocky Linux Lab

Learning enterprise Linux administration from the ground up — using Rocky Linux in a virtualised environment to explore the CLI, server configuration, and Linux fundamentals.

// about this lab

Why Rocky Linux?

Rocky Linux is a community-driven enterprise Linux distribution, designed as a downstream rebuild of Red Hat Enterprise Linux (RHEL). It's widely used in enterprise server environments, making it a practical choice for anyone looking to build skills in Linux administration.

This lab is the starting point — getting comfortable with the OS, the CLI, and how Linux fits into an enterprise infrastructure alongside Windows Server and Active Directory.
