Enterprise Infrastructure Lab

A simulated corporate IT environment built to mirror real-world enterprise systems — Active Directory, virtualised servers, secure file services, backup, and infrastructure monitoring.

// overview

Project Details

This lab demonstrates a full Active Directory setup including domain controllers, user accounts, security groups, and OU structure. Built using Windows Server 2022, VMware Workstation, and Microsoft Hyper-V to develop practical infrastructure and systems administration skills.

The goal is to simulate what a real enterprise IT environment looks like — so that when I step into a support or admin role, I've already navigated the same challenges.
