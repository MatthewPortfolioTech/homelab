Cloud Infrastructure Lab

Hands-on cloud environment covering virtual networks, compute resources, storage, identity services, and secure connectivity — focused on real-world deployment and operations.

// overview

Project Details

This lab will simulate an enterprise cloud environment across Azure and AWS. The goal is to understand how cloud infrastructure is provisioned, secured, and managed — building on the on-premises skills developed in the Enterprise Infrastructure lab.

Planned areas include virtual networking, VM deployment, storage accounts, identity management via Azure AD/AWS IAM, and monitoring with cloud-native tools.
