Linux Infrastructure Lab

A virtualised Linux environment built to explore server administration, remote access, file sharing, and basic infrastructure management using Ubuntu.

// overview

Project Details

This lab focuses on building practical Linux server administration skills in a virtualised environment using VMware. The goal is to get comfortable managing a Linux system from the command line — covering user management, file services, remote access, firewall configuration, and basic monitoring.

Linux is a critical skill for anyone moving into network or cloud engineering, and this lab is the foundation for understanding how enterprise Linux environments are set up and maintained.

// skills & tools

Technologies Used

Ubuntu Server VMware Workstation Linux CLI SSH Samba File Sharing UFW Firewall User & Group Management System Monitoring
// lab setup

Environment Setup

Ubuntu was installed and configured inside VMware Workstation. Initial setup included creating a user account, configuring the network, and getting familiar with the Linux terminal environment.
